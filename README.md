# CuorApe – Demo (Canvas)

Mini-sito one-file in HTML5 Canvas per il brand **CuorApe** (gelati artigianali su Apecar).  

## Funzioni
1. Pagina Filosofia Aziendale
2. Pagina con Elenco dei Gusti
3. Pagina dedicata agli Eventi
4. Pagina Contatti

## Grafica
Grafica Cartoon semplice e simpatica

## Struttura
- Il file `index.html` include tutte le funzioni (animazione, navigazione on-canvas, form con mailto).
- Il favicon è in formato SVG (`favicon.svg`).

## Note
La versione attuale è una demo, molte informazioni sono da sistemare, così come sono necessari ancora dei ritocchi grafici.

## Pubblicazione veloce

### Opzione A — GitHub Pages
1. Crea un repo su GitHub (es. `cuorape`).
2. Carica i file di questa cartella: `index.html` e `favicon.svg` (opzionale `README.md`).
3. Vai su **Settings → Pages → Source** e seleziona `main / root`, quindi **Save**.
4. L’URL sarà: `https://<tuo-utente>.github.io/cuo r a p e/`

### Opzione B — Netlify Drop
1. Apri https://app.netlify.com/drop
2. Trascina dentro il file `index.html` (e `favicon.svg`).
3. Riceverai subito un link pubblico tipo `https://qualcosa.netlify.app`.
